# AI Algorithms Repository

This repository contains implementations of artificial intelligence algorithms developed as part of the course **"Introduction to Artificial Intelligence"** at the Warsaw University of Technology.

## Overview

The project includes solutions to various assignments covering topics such as:

* State space search (e.g., A*, Best-First Search)
* Evolutionary and genetic algorithms
* Optimization problems (e.g., Rastrigin function)
* Game-playing algorithms (Minimax, Alpha-Beta Pruning)

## Purpose

The goal is to practice and compare different AI techniques and understand their performance in solving computational problems.
